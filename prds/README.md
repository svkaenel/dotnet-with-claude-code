# README

Dieser Ordner enthält Beispiel PRDs, die wir so verwendet haben und soll lediglich einen Eindruck von einer gut verarbeitbaren Komplexität eines PRD vermitteln. 

Die *-prompt-Dateien sind dabei der manuelle geschriebene Prompt des Entwicklers und
die *-generated-Dateien, die fertigen PRDs, die über das custom Command /generate-prd <prompt-file> erzeugt wurden.

Die Codierund durch Claude Code wird dann per /execute-prd <prd-file> angestoßen.
