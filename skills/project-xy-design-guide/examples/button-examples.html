Code Examples

Primary Button HTML:

<button class="bg-primary text-white px-6 py-3 !rounded-button hover:bg-opacity-90 transition whitespace-nowrap">
    Button Text
</button>

Secondary Button HTML:

<button class="bg-secondary text-white px-6 py-3 !rounded-button hover:bg-opacity-90 transition whitespace-nowrap">
    Call to Action
</button>

Outlined Button HTML:

<button class="border-2 border-primary px-6 py-3 !rounded-button hover:bg-primary hover:text-white transition whitespace-nowrap">
    Outlined Action
</button>

Button with Icon HTML:

<button class="bg-primary text-white px-6 py-3 !rounded-button hover:bg-opacity-90 transition flex items-center gap-2 whitespace-nowrap">
    <i class="ri-send-plane-line"></i>
    <span>Send Message</span>
</button>

Best Practices

  • Always use !rounded-button (8px radius) for consistent button styling
  • Include whitespace-nowrap to prevent text wrapping on buttons
  • Add transition class for smooth hover effects
  • Use hover:bg-opacity-90 for primary/secondary button hover states
  • Use hover:bg-primary hover:text-white for outlined button hover states
  • Add flex items-center gap-2 when combining text with icons
  • Use primary (#282824) for main actions, secondary (#BC002D) for CTAs
  • Include disabled attribute and opacity-70 cursor-not-allowed for disabled states
